LessCSS Converter 1.0 README

Author: Francesco Illuminati (fillumina@gmail.com)

This is a java program that needs the Java Standard Edition to be to be executed.
You can download and install the latest JSE for your platform from:

http://www.oracle.com/technetwork/java/javase/downloads/index.html

To execute the program you may simply click (or double click) on it, depending
from your operative system, or you may try to start it manually issuing the
following command from the directory the file is on:

java -jar LessCssConverter.jar
